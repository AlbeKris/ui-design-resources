# Насмотренность · Ресурсы для дизайна экранов, сайтов & UI

Карта из **80 ресурсов** для дизайна интерфейсов — от общего вдохновения до готовых компонентов, анимаций, цвета, шрифтов и инструментов. 9 разделов, у каждого ресурса — описание, тип и кликабельная ссылка.

## Открыть онлайн

https://albekris.github.io/ui-design-resources/

## Функции

- 🔍 Поиск по названию и описанию
- 🏷 Фильтр по типу: насмотренность / можно брать / инструмент / справочник
- 🧭 Навигация по 9 разделам
- ⭐ Избранное (сохраняется в браузере)

## Структура

- `index.html` — сайт-каталог (светлый минимализм), всё в одном файле

Сайт статический, без сборки — открывается двойным кликом и работает на GitHub Pages.
